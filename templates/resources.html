Getting Started

Anaconda Distribution

Get Python and the Anaconda distribution which includes almost all of the libraries you will need to get started.

IPython

Learn about IPython and IPython Notebooks.

Python For Data Mining

Read my introduction post on how to use Python for data mining.

Online Courses

Coursera Machine Learning

This excellent course is probably the most popular. Taught by Andrew Ng who is now the Chief Scientist at Baidu; great coverage of topics. I would definitely recommend starting here for online learning.

Harvard Data Science

I love this course because it uses Python. Also has a good coverage of topics and a great github repo of code. Great place to learn data science using Python.

Sites

Yhat Blog

The Yhat blog always seems to have awesome articles on data science. Great place to learn and see some cool projects.

DataTau

DataTau is like the Hacker News of data science. The site isn't nearly as active as Hacker News, but often has some nice posts. Would be cool to help it to continue to grow.

Mike Bostock

One of the developers of D3.js - this guy is the king of interactive visualization. Check out his stuff and be amazed. He also posts amazing tutorials.

While My MCMC Gently Samples

Thomas Wiecki is very skilled at writing tutorials. Focuses mostly on Python and Bayesian methods; best place to learn how to use Python for Bayesian statistics.

Podcasts

Partially Derivative

Great podcast that covers data science news of the week in a very entertaining fashion. Great way to keep up with data related stories.

Talking Machines

A podcast focused more on machine learning research. Some great interviews with leading researchers.

O'Reilly Data Show

Tends to focus on interviews with people from industry. Provides some great insights into how data science is being used in industry.

Books

An Introduction to Statistical Learning

If I were to pick a book to start with, this would be it. Great explanations of most of the important topics with R code. For a more advanced version, check out The Elements of Statistical Learning.

Applied Predictive Modeling

As stated, this is a very applied book with R code. Similar to Introduction to Statistical Learning, but hits more on certain topics such as data pre-processing.

Doing Bayesian Data Analysis

By far the best book I have read on how to do Bayesian Data Analysis that doesn't assume much prior knowledge. Tons of R code and hits on lots of great topics.

Bayesian Methods for Hackers

This is the best book I have seen on Bayesian methods using Python. I would recommend learning about Bayesian statistics from Doing Bayesian Data Analysis and then using this book to help learn how to do things in Python.

Python for Data Analysis

From the creator of Pandas, this is the best book to learn the Python language from a data science perspective. A lot of this information is in the online docs, but still a great resource with data analysis examples.

Interactive Data Visualization for the Web

By far the best resource I have found to learn how to use D3.js to make sweet visualizations. It assumes little prior knowledge about java script and CSS. Definitely check it out if you enjoy creating visualizations.
